# Git AI & xEo reference links

Quick links and install for Git AI (track AI code to production) and xEo (AI agent skills collection).

## Git AI (usegitai.com)

- **Site:** https://usegitai.com/
- **Prompt analysis (CLI):** https://usegitai.com/docs/cli/prompt-analysis  
  Analyze prompting practices: load prompts from local/store, grade by best practices, categorize by work type, correlate with acceptance rate. Skill: `/git-ai:prompt-analysis`. Works with Claude Code (add `~/.git-ai/skills` as plugin).
- **Install (Mac, Linux, Windows):**
  ```bash
  curl -sSL https://usegitai.com/install.sh | bash
  ```

Git AI is an open-source Git extension that tracks AI-generated code through the SDLC (Cursor, Claude Code, Copilot, Gemini, OpenCode, Continue, etc.), stores prompts, and supports `git-ai blame` and personal/team stats.

## xEo (AI Agent Skills Collection)

- **Repo:** https://github.com/GPTJunkie/xEo  

Human-vetted, community-curated skills for AI coding & agent tools. Originally for Claude Code; skills work with various AI agents. Apache 2.0 (some indexed skills have other licenses). Data stored in-repo for license consistency, integrity, and convenient access.

*Note: This is the “xEo” skills repo on GitHub, not the XEO/DEO (experience/performance optimization) section in [xeo.md](xeo.md).*
